DevQuery is a full-stack forum application where developers can register, log in, browse categorized topics (JavaScript, React, Node.js), ask questions, and post answers.

Built following the 3-tier architecture requirements of the Software Engineering Bootcamp **Project 4: Node.js + Express**.

## Features

- User registration and login (JWT authentication, bcrypt password hashing)
- Real-time form validation with error messages that disappear on input change
- Dashboard with scrollable category sidebar
- View questions in chronological order (newest first) per category
- Ask new questions and post answers
- Responsive layout (mobile-friendly sidebar collapse)
- Clean, modern UI with Tailwind CSS
- MongoDB database with seeded example data

## Tech Stack

**Backend**
- Node.js + Express.js
- MongoDB (Atlas)
- JWT for authentication
- bcryptjs for password hashing

**Frontend**
- React (Create React App)
- React Router v6
- Axios for API calls
- Tailwind CSS for styling

**Database**
- MongoDB Atlas (free tier)
